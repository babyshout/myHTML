주제 : 요리 레시피 (레시피 추천 사이트 겸 본인들이 한 레시피 올리는 커뮤니티 사이트)


메인화면 
	-로그인 정보(내정보)
	-메뉴 상단 출력
	-홈페이지 이름 출력
	-추천수 많은 레시피 홈화면 출력
메뉴

	-요리 추천
		- 계절 요리 추천(여름)
	-내가 한 요리 (이미지랑 레시피 설명 )	
		-블로그 처럼 게시
	-지금까지했던 레시피(사람들이 한 요리들)
		-인스타 피드처럼 이미지와 레시피 함께출력
	-좋아하는 요리(사람들의 요리 선호도)
		-사람들이 많이한 요리 순위
	-재료로 할 수 있는요리( hover속성으로 재료 이름 출력 )
		-닭
		-해물
		
회원가입 
	-이름 ,나이, 성별, 이메일, 아이디,비번 
	-회원가입 버튼, 취소(reset) 버튼
로그인화면 
	-아이디 적는칸 ,
	-비번 적는칸 ,
	- 로그인 버튼 

----------------------------------------------------------------------------------------
	1. 링크 들어갔을때 	사이트 이름 출력
		로그인(회원가입) 화면이 나오게끔
	2. 로그인(회원가입) 완료 후 메인 홈화면이 나옴
	3.홈 화면 기타 기능 출력(메뉴 5개 ) 
	4.메뉴 클릭 시 새로운 화면 으로 넘어감
